# 🔥 Stylish Temperature Converter 🌡️

A visually engaging temperature conversion app built using **Python's tkinter library**. This project converts temperature values between **Celsius**, **Fahrenheit**, and **Kelvin**, wrapped in a vibrant gradient UI with animated buttons for an interactive feel.![Screenshot 2025-06-27 151645](https://github.com/user-attachments/assets/11db1ac1-0561-44b0-8e43-c953f3c1ef91)


---

## ✨ Features

- Accepts signed integers (like `-10`, `+25`, or `0`) only
- Converts between all three major temperature units
- Hover-sensitive button with color transitions
- Custom gradient background simulating thermal flow
- Built-in input validation for clean user interaction
